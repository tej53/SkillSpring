# E-Commerce Platform Search Function

# This is a Java-based E-commerce Platform Search Function that helps users search products efficiently.

It allows users to:
- Store Products
- Search Products using Linear Search
- Search Products using Binary Search
- Display Product Details

Data Structures Used:
- Array

Algorithms Used:
- Linear Search
- Binary Search

Time Complexities:
- linearSearch : O(n)
- binarySearch : O(logn)
- display() : O(1)
