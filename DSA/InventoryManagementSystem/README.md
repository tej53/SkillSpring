# Inventory Management System

# Description
This is a Java-based Inventory Management System that manages the products in warehouse efficiently.

It allows users to:
- Add Products
- Update Products
- Delete Products
- Display Products

Data Structures Used: 
- HashMap

Time Complexities:
- addProduct() : O(1)
- updateProduct() : O(1)
- deleteProduct() : O(1)
- displayProducts() : O(n)
