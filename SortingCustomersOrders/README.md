# Sorting Customer Orders

# Description

This Java project sorts customer orders based on total price.

It allows users to:
- Store customer orders
- Sort orders using Bubble Sort
- Sort orders using Insertion Sort
- Sort orders using Quick Sort
- Display sorted orders


Algorithms Used:
- Bubble Sort
- Insertion Sort
- Quick Sort


Time Complexities:

Bubble Sort:
- Best Case : O(n)
- Average Case : O(n²)
- Worst Case : O(n²)


Insertion Sort:
- Best Case : O(n)
- Average Case : O(n²)
- Worst Case : O(n²)


Quick Sort:
- Best Case : O(n log n)
- Average Case : O(n log n)
- Worst Case : O(n²)


Function Time Complexities:

- bubbleSort() : O(n²)
- insertionSort() : O(n²)
- quickSort() : O(n log n)


Best Approach:
Quick Sort is preferred because it is faster for large datasets and uses the divide and conquer technique.
