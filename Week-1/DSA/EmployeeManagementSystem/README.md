# Employee Management System

# Description

This is a Java-based Employee Management System that manages employee records efficiently using arrays.

It allows users to:
- Add Employees
- Search Employees
- Display Employees
- Delete Employees

Data Structure Used:
- Array

Time Complexities:
- addEmployee() : O(1)
- searchEmployee() : O(n)
- traverseEmployees() : O(n)
- deleteEmployee() : O(n)

Limitations:
- Fixed size
- Deletion requires shifting elements
- Insertion in the middle is costly

Best Use Case:
Arrays are suitable for applications where the number of records is fixed and fast index-based access is required.
