# Financial Forecasting

# Description

This is a Java-based Financial Forecasting application that predicts future investment values using a recursive approach.

It allows users to:

* Calculate Future Value
* Predict Investment Growth
* Forecast Returns Based on Growth Rate
* Display Forecast Results

Algorithm Used:

* Recursion

Time Complexities:

* forecast() : O(n)

Space Complexity:

* forecast() : O(n)
